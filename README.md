# Login dan Register Sederhana Menggunakan Laravel

Sistem autentikasi sederhana berbasis Laravel yang mendukung fitur login, registrasi, reset password melalui email, serta proteksi halaman menggunakan middleware authentication.

## 🚀 Fitur

* Login User
* Registrasi User
* Logout
* Forgot Password
* Reset Password via Email
* SMTP Gmail Integration
* Password Hashing
* Middleware Authentication
* Expired Reset Link Protection

## 🛠️ Tech Stack

* Laravel 12
* PHP 8.3
* MySQL
* Tailwind CSS
* Vite
* Laragon

## ⚙️ Instalasi

Clone repository:

```bash
git clone https://github.com/hikmaldwi2021-hue/Login-dan-Register-Sederhana-Menggunakan-Framework-Laravel.git
```

Masuk ke folder project:

```bash
cd Login-dan-Register-Sederhana-Menggunakan-Framework-Laravel
```

Install dependency:

```bash
composer install
npm install
```

Salin file environment:

```bash
copy .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Konfigurasi database pada file `.env`, kemudian jalankan:

```bash
php artisan migrate
```

Jalankan Vite:

```bash
npm run dev
```

Jalankan server Laravel:

```bash
php artisan serve
```
